Configuration requise

  • Windows
  • Android
  • iOS
  • Configuration minimal

    • Système d'exploitation : Windows 7 SP1 64 bits, Windows 8.1 64 bits ou Windows 10 64 bits
    Processeur : Intel Core i5 ou équivalent
    RAM : 8 Go
    Carte graphique : NVIDIA® GeForce® GT 1030 ou supérieure
    Version DirectX : DirectX 11
    Espace de stockage : au moins 40 Go d'espace libre

  • Configuration recommandée

    • Système d'exploitation : Windows 7 SP1 64 bits ou Windows 8.1 64 bits ou Windows 10 64 bits
    Processeur : Intel Core i7 ou équivalent
    RAM : 16 Go
    Carte graphique : NVIDIA® GeForce® GTX 1060 6 Go ou supérieure
    Version DirectX : DirectX 11
    Espace de stockage : au moins 40 Go d'espace libre
    Si votre PC ne respecte pas la configuration ci-dessus, il vous sera impossible d'installer ou d'exécuter le jeu. Votre écran peut également se figer ou devenir noir.

  • Configuration compatible

    • Compatibilité requise : Architecture Arm v8a 64 bits
    Processeur sans PowerVR
    RAM : plus de 3 Go
    Système : Android 7.0 ou supérieur
    Espace de stockage : au moins 10 Go

  • Configuration recommandée

    • Processeur : Snapdragon 865, Kirin 820
    RAM : plus de 4 Go

  • Espace de stockage :

    • au moins 20 Go

  • Système compatible :

    • iOS 9.0 et supérieur
    (Compatibilité manette Bluetooth : iOS 14 et supérieur)
